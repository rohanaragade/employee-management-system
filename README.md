# Employee Management System

Developed a secure Employee Management System using Spring Boot, Spring Security, Hibernate, JPA, and MySQL with CRUD operations and authentication for protected REST APIs.

## Technologies Used
- Java
- Spring Boot
- Spring Security
- Spring MVC
- Spring Data JPA
- Hibernate
- MySQL
- Postman

## Features
- Add Employee
- Update Employee
- Delete Employee
- Fetch Employee Details
- Secure REST APIs
- Authentication & Authorization

## Architecture
- Controller Layer
- Service Layer
- Repository Layer
